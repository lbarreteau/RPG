# My-RPG

Foobar is a Python library for dealing with word pluralization.

## Installation

Download the repository package:

```bash
git clone git@github.com:lbarreteau/RPG.git
```

To compile the project

```bash
Make
```
## Usage

```bash
./my_rpg
```
